# Project Brief — MercApp

Fecha: 2025-11-04

Resumen:
MercApp es una aplicación mobile-first para registrar y gestionar compras de supermercado. Permite a los usuarios crear cuentas, registrar compras y productos, escanear códigos de barras, y revisar historial y estadísticas de gastos.

Objetivo principal:
- Construir una MVP backend en TypeScript (Express + Prisma + PostgreSQL) y un frontend en React + Vite con enfoque mobile-first.

Alcance inicial (MVP):
- Registro/Login (JWT + bcrypt)
- Registrar compra y agregar productos
- Escaneo básico de códigos de barras
- Historial de compras y detalle
